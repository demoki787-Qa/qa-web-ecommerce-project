Projeto de Teste - Sistema Web E-commerce

Responsavel: Demoki

Objetivo:
Validar as principais funcionalidades de um sistema de loja virtual.

Funcionalidades testadas:
- Cadastro de usuário
- Login
- Busca de produtos
- Adição ao carrinho
- Finalização de compra

Tipos de teste aplicados:
- Teste Funcional
- Teste Negativo
- Teste de Usabilidade

Validação de Banco de Dados:

Durante os testes foram utilizadas queries SQL para validar a integridade
dos dados no banco de dados, 
verificando inconsistências como:
- usuários duplicados
- pedidos sem associação com usuário
- produtos com preço inválido

  Status do Projeto:
Projeto finalizado com documentação de testes, bug reports,
evidências e validações SQL.
